# Linda Estética - Versão GitHub Pages

Landing page otimizada para GitHub Pages (apenas frontend).

## Deploy Rápido

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/SEU-USUARIO/linda-estetica.git
git push -u origin main
```

Depois: Settings → Pages → Source: main → Save

Acesse: `https://SEU-USUARIO.github.io/linda-estetica/`

## Configurar Formulário

1. Acesse: https://formspree.io
2. Crie conta e novo formulário
3. No `index.html`, substitua `SEU_FORM_ID` pelo ID real

Veja `DEPLOY_GITHUB_PAGES.md` para guia completo.
